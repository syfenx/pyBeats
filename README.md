# pyBeats
Python Beat Sequencer

Windows and Linux (Linux preferred)

Python 3.4

Dependencies:
- Pygame
- Pybass

Shortcuts:
- numbers 1 - 0 (not numpad) are basic sounds / samples, right click to add, left click to delete.
- mousewheel over note controls volume / velocity
